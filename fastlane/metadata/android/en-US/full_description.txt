<p><i>Note Calendar</i> is a customizable offline app which allows you to manage notes for each day and offers a way to quickly reach for each of them. App does not contain ads.</p><p><br><b>App possibilities:</b></p><p>Day tab presents selected day information and note assigned to that day. It also provides a straightforward way to add, edit and delete note.</p><p>Week tab holds a list of days to quickly move to other day of the week.</p><p>You can easily see which days contain note via month tab. If a number of the day of the mounth is surrounded by a ring, then it contains a note.</p><p>Application offers notifications which can be enabled to remind you of a note created on the next day at a given time. For example you can set notification time at time before your alarm clock time in order to wake up and read note about something important to do.</p><p><br><b>Settings tab includes:</b></p><ul><li>Theming:<ul><li>Setting custom theme which includes 10 modifiable colors</li><li>Setting light theme</li><li>Setting dark theme</li><li>Setting default theme based on system settings</li></ul></li><li>Notes deletion</li><li>Notifications management:<ul><li>Enabling or disabling</li><li>Setting notification time</li></ul></li><li>Setting first day of week</li></ul>
